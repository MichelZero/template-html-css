💡 1. Formatação Semântica vs Visual

🎯 Conceito Fundamental

HTML deve expressar o significado do conteúdo, não apenas sua aparência visual. Use CSS para styling!

✨ Elementos Semânticos de Formatação

<!-- ELEMENTOS SEMÂNTICOS (recomendados) -->

<!-- Ênfase (geralmente itálico) -->
<p>Este texto tem <em>ênfase especial</em> na frase.</p>

<!-- Importância (geralmente negrito) -->
<p>Esta informação é <strong>muito importante</strong>!</p>

<!-- Texto marcado/destacado -->
<p>Encontre a palavra <mark>destacada</mark> no texto.</p>

<!-- Texto pequeno (nota de rodapé, direitos autorais) -->
<p><small>© 2024 Empresa. Todos os direitos reservados.</small></p>

<!-- Texto deletado/removido -->
<p>O preço era <del>R$ 100</del> R$ 80.</p>

<!-- Texto inserido/adicionado -->
<p>Novo recurso: <ins>suporte a pagamentos via PIX</ins>.</p>

<!-- Subscrito e sobrescrito -->
<p>Fórmula da água: H<sub>2</sub>O</p>
<p>Einstein: E = mc<sup>2</sup></p>

⚠️ Evite Elementos Puramente Visuais

<!-- NÃO RECOMENDADO (apenas visual) -->
<b>Texto em negrito</b>
<i>Texto em itálico</i>
<u>Texto sublinhado</u>

<!-- RECOMENDADO (semântico) -->
<strong>Texto importante</strong>
<em>Texto com ênfase</em>
<mark>Texto destacado</mark>

🎨 Outros Elementos de Formatação

<!-- Abreviações -->
<p>O <abbr title="HyperText Markup Language">HTML</abbr> é uma linguagem de marcação.</p>

<!-- Definições -->
<p><dfn>HTML</dfn> é uma linguagem de marcação para criar páginas web.</p>

<!-- Endereços -->
<address>
    João Silva<br>
    Rua das Flores, 123<br>
    São Paulo - SP
</address>

<!-- Data e hora -->
<p>Publicado em <time datetime="2024-01-15">15 de janeiro de 2024</time>.</p>

📋 2. Listas em HTML

Listas são fundamentais para organizar informações. HTML oferece três tipos principais:

📝 Lista Não Ordenada (ul)

<h3>Ingredientes para o bolo:</h3>
<ul>
    <li>2 xícaras de farinha</li>
    <li>3 ovos</li>
    <li>1 xícara de açúcar</li>
    <li>200ml de leite</li>
</ul>

🔢 Lista Ordenada (ol)

<h3>Modo de preparo:</h3>
<ol>
    <li>Pré-aqueça o forno a 180°C</li>
    <li>Misture os ingredientes secos</li>
    <li>Adicione os ovos e o leite</li>
    <li>Asse por 30 minutos</li>
</ol>

<!-- Lista ordenada personalizada -->
<ol start="5">
    <li>Deixe esfriar</li>
    <li>Desenforms e sirva</li>
</ol>

<!-- Tipos de numeração -->
<ol type="A">
    <li>Primeira opção</li>
    <li>Segunda opção</li>
</ol>

<ol type="I">
    <li>Números romanos</li>
    <li>Também funcionam</li>
</ol>

📖 Lista de Definição (dl)

<h3>Glossário de Termos Web:</h3>
<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language - linguagem de marcação para web</dd>
    
    <dt>CSS</dt>
    <dd>Cascading Style Sheets - linguagem para estilização</dd>
    
    <dt>JavaScript</dt>
    <dd>Linguagem de programação para interatividade</dd>
</dl>

💡 Dicas para Listas

  • Use ul quando a ordem não importa
  • Use ol para sequências e passos
  • Use dl para glossários e pares termo-definição
  • Evite listas com apenas um item
  • Cada <li> pode conter qualquer elemento HTML

💻 4. Código e Pré-formatação

🔧 Código Inline

<p>
    Use o elemento <code>&lt;strong&gt;</code> para destacar texto importante.
    Para variáveis em programação, use <var>nomeDaVariavel</var>.
</p>

<p>
    Para mostrar entrada do teclado: <kbd>Ctrl + C</kbd> para copiar.
    Para saída do sistema: <samp>Erro 404: Página não encontrada</samp>.
</p>

📋 Bloco de Código (pre)

<pre>
<code>
function saudar(nome) {
    return "Olá, " + nome + "!";
}

console.log(saudar("Mundo"));
</code>
</pre>

🎨 Código com Linguagem

<pre>
<code class="language-javascript">
// Exemplo de função em JavaScript
const calcularIdade = (anoNascimento) => {
    const anoAtual = new Date().getFullYear();
    return anoAtual - anoNascimento;
};
</code>
</pre>

<pre>
<code class="language-html">
&lt;!DOCTYPE html&gt;
&lt;html lang="pt-BR"&gt;
&lt;head&gt;
    &lt;meta charset="UTF-8"&gt;
    &lt;title&gt;Minha Página&lt;/title&gt;
&lt;/head&gt;
&lt;/html&gt;
</code>
</pre>

🎯 Quando Usar Cada Elemento

  • <code> - Código inline curto
  • <pre> - Preservar formatação (espaços, quebras)
  • <var> - Variáveis e placeholders
  • <kbd> - Entrada do teclado
  • <samp> - Saída do sistema

🚀 5. Elementos de Texto Avançados

📐 Detalhes e Sumário

<details>
    <summary>FAQ: Como aprender HTML rápido?</summary>
    <p>
        A melhor forma é praticar criando páginas reais. Comece com projetos 
        simples como uma página pessoal, depois evolua para layouts mais complexos.
    </p>
    <ul>
        <li>Pratique diariamente</li>
        <li>Valide seu código</li>
        <li>Estude exemplos reais</li>
    </ul>
</details>

<details open>
    <summary>Esta seção já vem aberta</summary>
    <p>Use o atributo <code>open</code> para mostrar o conteúdo inicialmente.</p>
</details>

🎯 Figura e Legenda

<figure>
    <pre>
    <code>
&lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;head&gt;
    &lt;title&gt;Hello World&lt;/title&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;h1&gt;Hello, World!&lt;/h1&gt;
&lt;/body&gt;
&lt;/html&gt;
    </code>
    </pre>
    <figcaption>
        Exemplo 1: Estrutura básica de uma página HTML
    </figcaption>
</figure>

🔄 Direção de Texto

<!-- Texto bidirecional -->
<p>Este é um texto normal com <bdo dir="rtl">este trecho da direita para esquerda</bdo>.</p>

<!-- Isolamento bidirecional -->
