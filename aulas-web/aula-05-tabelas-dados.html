AULA 5

📊 Tabelas e Dados

Aprenda a estruturar dados de forma acessível e profissional usando tabelas HTML, com foco em semântica, responsividade e usabilidade.

⏱️ 50 minutos
🎯 Intermediário
📝 4 Exercícios

🎯 Objetivos de Aprendizagem

  • Entender a estrutura semântica de tabelas
  • Criar tabelas acessíveis com cabeçalhos adequados
  • Implementar agrupamento de dados com tbody, thead e tfoot
  • Aplicar responsividade em tabelas
  • Usar atributos de escopo e acessibilidade

🏗️ 1. Estrutura Básica de Tabelas

🎯 Quando Usar Tabelas

Tabelas devem ser usadas apenas para dados tabulares - informações que fazem sentido em linhas e colunas. Nunca use tabelas para layout da página.

📋 Elementos Fundamentais

<!-- Tabela básica -->
<table>
    <tr>
        <th>Nome</th>
        <th>Idade</th>
        <th>Profissão</th>
    </tr>
    <tr>
        <td>Ana Silva</td>
        <td>28</td>
        <td>Desenvolvedora</td>
    </tr>
    <tr>
        <td>Carlos Santos</td>
        <td>35</td>
        <td>Designer</td>
    </tr>
</table>

🎯 Estrutura Semântica Completa

<table>
    <caption>Funcionários da Empresa - Q1 2024</caption>
    
    <thead>
        <tr>
            <th scope="col">ID</th>
            <th scope="col">Nome Completo</th>
            <th scope="col">Departamento</th>
            <th scope="col">Salário</th>
        </tr>
    </thead>
    
    <tbody>
        <tr>
            <td>001</td>
            <td>Maria Fernanda Costa</td>
            <td>Tecnologia</td>
            <td>R$ 8.500</td>
        </tr>
        <tr>
            <td>002</td>
            <td>João Pedro Oliveira</td>
            <td>Marketing</td>
            <td>R$ 6.200</td>
        </tr>
        <tr>
            <td>003</td>
            <td>Patricia Lima</td>
            <td>Financeiro</td>
            <td>R$ 7.800</td>
        </tr>
    </tbody>
    
    <tfoot>
        <tr>
            <td colspan="3"><strong>Total Salários:</strong></td>
            <td><strong>R$ 22.500</strong></td>
        </tr>
    </tfoot>
</table>

⚠️ Elementos Obrigatórios

  • <table>: Container principal da tabela
  • <tr>: Linha da tabela (table row)
  • <th>: Célula de cabeçalho (table header)
  • <td>: Célula de dados (table data)
  • <caption>: Título/descrição da tabela

💡 Dicas de Acessibilidade

  • Sempre use <caption> para descrever o propósito da tabela
  • Use scope="col" e scope="row" corretamente
  • Para tabelas complexas, use headers attribute
  • Mantenha estrutura lógica de leitura

📱 4. Tabelas Responsivas

🔄 Scroll Horizontal

<div class="table-container">
    <table class="table-responsive">
        <caption>Dados de Vendas - Role horizontalmente para ver mais</caption>
        <thead>
            <tr>
                <th scope="col">Produto</th>
                <th scope="col">Jan</th>
                <th scope="col">Fev</th>
                <th scope="col">Mar</th>
                <th scope="col">Abr</th>
                <th scope="col">Mai</th>
                <th scope="col">Jun</th>
                <th scope="col">Total</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <th scope="row">Notebook Dell</th>
                <td data-label="Jan">45</td>
                <td data-label="Fev">52</td>
                <td data-label="Mar">48</td>
                <td data-label="Abr">55</td>
                <td data-label="Mai">60</td>

                <td data-label="Total"><strong>318</strong></td>
            </tr>
        </tbody>
    </table>
</div>

📱 CSS para Responsividade

/* Container para scroll horizontal */
.table-container {
    overflow-x: auto;
    -webkit-overflow-scrolling: touch;
    border: 1px solid #ddd;
    border-radius: 8px;
}

.table-responsive {
    min-width: 600px;
    width: 100%;
    border-collapse: collapse;
}

/* Estilo mobile-first */
@media (max-width: 768px) {
    .table-responsive,
    .table-responsive thead,
    .table-responsive tbody,
    .table-responsive th,
    .table-responsive td,
    .table-responsive tr {
        display: block;
    }
    
    .table-responsive thead tr {
        position: absolute;
        top: -9999px;
        left: -9999px;
    }
    
    .table-responsive tr {
        border: 1px solid #ccc;
        margin-bottom: 10px;
        padding: 10px;
        border-radius: 8px;
    }
    
    .table-responsive td {
        border: none;
        padding: 10px;
        text-align: right;
        padding-left: 50%;
        position: relative;
    }
    
    .table-responsive td:before {
        content: attr(data-label) ": ";
        position: absolute;
        left: 10px;
        width: 45%;
        text-align: left;
        font-weight: bold;
    }
}

🎨 5. Estilização e UX

🎯 CSS Base para Tabelas

/* Estilos base */
table {
    border-collapse: collapse;
    width: 100%;
    margin: 20px 0;
    font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
}

caption {
    caption-side: top;
    text-align: left;
    font-weight: bold;
    font-size: 1.2em;
    margin-bottom: 10px;
    color: #333;
}

th, td {
    text-align: left;
    padding: 12px 15px;
    border-bottom: 1px solid #ddd;
}

th {
    background-color: #f8f9fa;
    font-weight: 600;
    color: #495057;
    border-top: 2px solid #007bff;
}

/* Zebra striping */
tbody tr:nth-child(even) {
    background-color: #f8f9fa;
}

/* Hover effects */
tbody tr:hover {
    background-color: #e8f4fd;
    cursor: default;
}

/* Focus para acessibilidade */
th:focus, td:focus {
    outline: 2px solid #007bff;
    outline-offset: -2px;
}
