📝 1. Briefing do Projeto Final

Seu desafio é criar um site de portfólio ou landing page para um serviço/produto fictício. O site deve ser:

  • 100% responsivo (mobile-first)
  • Utilizar animações e transições CSS
  • Organizado com metodologia BEM ou SMACSS
  • Otimizado para performance (Critical CSS, lazy loading)
  • Integrar interações com JavaScript (menu, animações, formulários)
  • Seguir boas práticas de acessibilidade

🏗️ 2. Estrutura Recomendada

  • Header: Logo, navegação responsiva (menu hamburger)
  • Hero: Imagem, título, chamada para ação
  • Sobre: Texto sobre você ou o produto
  • Projetos/Serviços: Grid responsivo de cards
  • Formulário de contato: Validação e feedback visual
  • Footer: Links de redes sociais, copyright

✅ 3. Checklist de Implementação

💡 4. Dicas para um Projeto Profissional

  • Planeje o layout antes de codar (wireframe)
  • Use variáveis CSS para cores e espaçamentos
  • Comente e documente seu código
  • Teste em diferentes dispositivos e navegadores
  • Otimize imagens e fontes
  • Utilize ferramentas de análise de performance
  • Priorize acessibilidade (alt em imagens, contraste, navegação por teclado)
